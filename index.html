About me

Hi! I'm an aspiring software developer passionate about building for the web and mobile platforms.

I enjoy working with frameworks like React, Vue, and Laravel for web development, as well as React Native, Expo, and Flutter for mobile apps.

I'm also deeply interested in computer servicing and Linux configuration. Linux is my daily driver—currently Void Linux—and I love fine-tuning my system to suit my workflow. Check out my Linux distro log.

Outside of software, I spend time exploring with electronics and programming microcontrollers using Arduino and MicroPython. It's a great way to connect software with hardware in creative ways.

I also like tinkering with random things and trying to fix them — mostly out of curiosity, even when I'm not sure I can.
